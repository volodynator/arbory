# Notion Clone

This repository contains a local development scaffold combining Vite, React, and Tauri for building a desktop/web application.

## Quick start

- Install dependencies:

```bash
npm install
```

- Start web development server:

```bash
npm run dev
```

- Build web bundle:

```bash
npm run build
```

- Build desktop app (Tauri):

```bash
npm run tauri:build
```
